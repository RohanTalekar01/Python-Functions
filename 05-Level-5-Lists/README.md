# 📘 Python Functions – Level 5: List Operations

## 📌 Description

This folder contains Python programs that demonstrate common list operations using functions. The programs focus on performing calculations, searching, sorting, modifying, and manipulating lists while emphasizing clean, reusable, and modular code.

These exercises strengthen problem-solving skills and provide hands-on practice with one of Python's most widely used data structures.

---

## 📂 Programs Included

| No. | Program |
|----|-------------------------------------------|
| 1 | Find the Sum of a List |
| 2 | Find the Maximum Element |
| 3 | Find the Minimum Element |
| 4 | Remove Duplicates from a List |
| 5 | Reverse a List |
| 6 | Sort a List Without Using `sort()` |
| 7 | Find the Second Largest Number |
| 8 | Merge Two Lists |
| 9 | Find Common Elements in Two Lists |
| 10 | Rotate a List by *k* Positions |

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/RohanTalekar01/Python-Functions.git
```

Navigate to this folder:

```bash
cd Python-Functions/Level-5-Lists
```

Run any program:

```bash
python Sum_of_List.py
```

or

```bash
python Rotate_List.py
```
