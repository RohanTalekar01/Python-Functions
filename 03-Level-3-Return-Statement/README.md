# 📘 Python Functions – Level 3 (Return)

## 📌 Description

This folder contains **10 Python programs** that focus on **functions with return values**. Unlike functions that only print output, these programs demonstrate how to use the `return` statement to send results back to the caller for further processing.

The exercises cover mathematical operations, number manipulation, and list-based calculations, helping you build a strong foundation in writing reusable and efficient Python functions.

---

# 🎯 Learning Objectives

By completing these exercises, you will learn how to:

- Use the `return` statement effectively
- Differentiate between `print()` and `return`
- Store returned values in variables
- Reuse returned results in other calculations
- Solve mathematical and list-based problems
- Write clean, modular, and reusable Python code

---

# 📂 Programs Included

| No. | Program | Description |
|----|--------------------------------|-------------------------------------------------------------|
| 1 | Square of a Number | Return the square of a given number. |
| 2 | Cube of a Number | Return the cube of a given number. |
| 3 | Even or Odd | Return whether a number is even or odd. |
| 4 | Factorial of a Number | Return the factorial of a given number. |
| 5 | Sum of Digits | Return the sum of all digits in a number. |
| 6 | Reverse a Number | Return the reversed form of a number. |
| 7 | Count Digits | Return the total number of digits in a number. |
| 8 | Largest Element in a List | Return the largest value from a list. |
| 9 | Smallest Element in a List | Return the smallest value from a list. |
| 10 | Average of a List | Return the average of all elements in a list. |
