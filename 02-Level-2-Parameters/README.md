# 📘 Python Functions – Level 2 (Parameters)

## 📌 Description

This folder contains **10 Python programs** that focus on **functions with parameters**. These exercises introduce the concept of passing values (arguments) to functions, allowing you to write reusable, modular, and efficient code.

The programs cover basic arithmetic operations, comparison-based logic, variable manipulation, and mathematical formula calculations. They are designed to strengthen your understanding of parameterized functions and improve your problem-solving skills.

---

# 🎯 Learning Objectives

By completing these exercises, you will learn how to:

* Define functions with parameters
* Pass arguments while calling functions
* Perform arithmetic operations using functions
* Use conditional statements for decision making
* Compare numbers using logical operators
* Write clean, reusable, and modular Python code
* Apply mathematical formulas through functions

# 📂 Programs Included

| No. | Program                    | Description                                                        |
| --- | -------------------------- | ------------------------------------------------------------------ |
| 1   | Add Two Numbers            | Calculate the sum of two numbers using a function.                 |
| 2   | Subtract Two Numbers       | Find the difference between two numbers.                           |
| 3   | Multiply Two Numbers       | Multiply two numbers using function parameters.                    |
| 4   | Divide Two Numbers         | Perform division and handle basic arithmetic operations.           |
| 5   | Largest of Two Numbers     | Determine the greater number between two values.                   |
| 6   | Smallest of Two Numbers    | Find the smaller number between two values.                        |
| 7   | Largest of Three Numbers   | Compare three numbers and return the largest.                      |
| 8   | Swap Two Numbers           | Exchange the values of two variables.                              |
| 9   | Simple Interest Calculator | Calculate Simple Interest using the formula **(P × R × T) / 100**. |
| 10  | Area of Rectangle          | Calculate the area using the formula **Length × Breadth**.         |

# ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/RohanTalekar01/Python-Functions.git
```

### Navigate to the project folder

```bash
cd Python-Functions/Level-2-Parameters
```

### Run any program

```bash
python Add_Two_Numbers.py
```

or

```bash
python Largest_of_Three.py
```
