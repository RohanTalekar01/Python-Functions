# 📘 Python Functions – Level 4: Logic Building

## 📌 Description

This folder contains Python programs that focus on solving common logic-building problems using functions. The programs cover number-based algorithms, string manipulation, and character analysis to strengthen problem-solving skills and improve understanding of Python functions.

Each program is implemented as a separate solution, emphasizing clean code, modular programming, and practical use of functions.

---

## 📂 Programs Included

| No. | Program |
|----|-------------------------------------------|
| 1 | Check if a Number is Prime |
| 2 | Check if a Number is an Armstrong Number |
| 3 | Check if a Number is a Palindrome |
| 4 | Check if a Year is a Leap Year |
| 5 | Check if a String is a Palindrome |
| 6 | Count Vowels in a String |
| 7 | Count Consonants in a String |
| 8 | Count Uppercase and Lowercase Letters |
| 9 | Count Words in a Sentence |
| 10 | Find the Frequency of Each Character |

## ▶️ Getting Started

Clone the repository:

```bash

```

```bash
cd Python-Functions/Level-4-Logic-Building
```

