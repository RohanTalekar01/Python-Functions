# 📘 Python Functions - Level 1 (Easy)

## 📌 Overview

This repository contains beginner-friendly Python function programs designed to build a strong foundation in function creation, function calls, loops, and basic logic building.

Each program focuses on solving a single problem using functions, making it easier to understand reusable code and modular programming.

---

## 🎯 Learning Objectives

By completing these exercises, you will learn how to:

* Define and call functions
* Understand function syntax
* Use loops inside functions
* Pass arguments to functions
* Build reusable code
* Improve logical thinking
* Write clean and readable Python programs

---

## 📂 Programs Included

| No. | Program                    | Concepts Used           |
| --- | -------------------------- | ----------------------- |
| 1   | Print "Hello World"        | Function Definition     |
| 2   | Print Your Name            | Function                |
| 3   | Print Numbers from 1 to 10 | Loop, Function          |
| 4   | Print Even Numbers (1–50)  | Loop, Condition         |
| 5   | Print Odd Numbers (1–50)   | Loop, Condition         |
| 6   | Multiplication Table       | Function with Parameter |
| 7   | Countdown from 10 to 1     | Loop                    |
| 8   | Squares from 1 to 10       | Loop, Arithmetic        |
| 9   | Cubes from 1 to 10         | Loop, Arithmetic        |
| 10  | First 20 Natural Numbers   | Loop                    |
